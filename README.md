Glosario de python:

`input("Entre")` : función que permite que el usario entre un dato por medio del teclado. Devuelve el valor entrado por el usuario para que pueda ser almacenado en una variable.

`print` : función que imprime a pantalla el valor la(s) variable(s) y/o literale(s) especificados.

`if` : palabra clave que permite la ejecución condicional de un bloque de código si se cumple la condición que le acompaña.

`else` : palabra clave que sigue a un bloque `if` o `elif` . Le sigue el bloque de código que será ejecutado si no se han cumplido las condiciones que acompañan al bloque `if` (y `elif`).

`elif` : palabra clave que sigue a un bloque `if` o `elif`. Le sigue el bloque de código que será ejecutado si **no** se han cumplido las condiciones que acompañan al bloque `if` y **si** se cumple la condición que acompaña al bloque elif.

`while`: palabra clave que permite repetir un bloque de código siempre y cuando se cumpla la condición que acompaña al `while`

`for var in List:` : palabra clave que permite repetir un bloque de código para todos elementos de una lista. En cada iteración la variable `var` asume un valor de la lista `List`

`def` : palabra clave que permite definir una función.
